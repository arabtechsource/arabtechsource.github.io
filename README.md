# Tech Blog Landing (RTL) — GitHub Pages

هذه نسخة تسويقية ثابتة (Static) لمدونة تقنية معلومات، بنفس ألوان القالب الحالي، وبدون صور/أيقونات.

## النشر على GitHub Pages
1) أنشئ Repo جديد باسم: `USERNAME.github.io`
2) ارفع محتويات هذا الـ ZIP (index.html + assets) إلى جذر المستودع.
3) من Settings → Pages اختر Branch: `main` و Folder: `/root`
4) بعد دقائق يفتح: https://USERNAME.github.io
